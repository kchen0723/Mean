# Mean
Sample for mean stack

Steps to setup:
1) install git and tortoise git tool
2) download nodejs and install it. nodejs also include npm
3) npm install -g angular
4) npm istall -g typescript
5) ng new VisualizeClient to create client project
6) make a VisualizeServer folder
7) go to VisualizeServer folder
8) npm init to create package.json
9) tsc --init to create ts.config
